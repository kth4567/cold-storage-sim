冷库屋面白天直接暴露在阳光下,吸收的太阳辐射把外表面晒得远高于气温(夏季深色屋面可达 60~70℃), 再通过顶板漏进库内。这解释了一个常见现象:冷库外表面尤其是屋面,多刷成白色——白色对太阳短波辐射的 吸收率低,能显著减少辐射得热。

物理图景

  • 任何温度高于绝对零度的物体都在向外发射热辐射(电磁波)。太阳(≈5800K)发的是可见光/近红外短波; 常温物体(≈300K)发的是长波红外——两个波段,各管各的。
  • 屋面同时在做三件事:吸收太阳短波 + 与天空交换长波 + 向空气对流散热, 综合结果决定了屋面外表面有多热。
  • 颜色的作用:对太阳短波的吸收率 α_s 由表面性质决定——白色/光亮金属 0.2~0.4, 黑色/深色 0.9 以上。低吸收 → 屋面更凉 → 漏热更少。
  • 辐射与导热、对流的本质区别:不需要介质,真空也能传,且热流量正比于温度的四次方。
一句话记忆:辐射不靠介质、正比于 T⁴、受表面颜色(吸收率/发射率)强烈影响。

① 斯特藩–玻尔兹曼定律(黑体辐射)

E_b = \sigma T^4, \qquad \sigma = 5.67\times10^{-8}\;\mathrm{W/(m^2{\cdot}K^4)}

⚠ T 必须用热力学温度(开尔文)!而且是四次方——温度稍升,辐射剧增。

② 实际物体(灰体)与基尔霍夫定律

E = \varepsilon\,\sigma T^4, \qquad 0 \le \varepsilon \le 1;\qquad \alpha_\lambda = \varepsilon_\lambda

ε 是发射率(黑体=1,抛光金属可低至 0.05);热平衡下同一波长处吸收率=发射率。 理想屋面是"选择性表面":对太阳短波低吸收(α_s 小)、对长波高发射(ε 大)——白天少吸热,夜里多散热。

③ 两表面间净辐射(小物体对大环境)

Q = \varepsilon\,\sigma A\,(T_{surf}^4 - T_{sky}^4)

④ 工程主角:太阳综合温度(sol-air temperature)

t_{sa} = t_o + \frac{\alpha_s\, I}{h_o} \qquad\Longrightarrow\qquad q = K\,(t_{sa} - t_i)

把太阳得热折算成一个"虚拟更高的室外温度",让外表面换热像纯对流一样处理—— 与 KP01 的 K 值模型无缝衔接。α_s 越小(越白),太阳当量温升 α_sI/h_o 越小。

拖动太阳走一天,把屋面从黑刷成白 直接拖太阳或点"播放一天"

例题:刷白到底值多少度?

白屋面 α_s=0.3 对比深色屋面 α_s=0.9, 太阳辐照 I=800 W/m²,外表面 h_o=20 W/(m²·K),忽略长波修正。比较两者的太阳当量温升。 (实验节正午参数即本题)

1
当量温升公式:
\Delta t_{sun} = \frac{\alpha_s\, I}{h_o}
2
白色屋面:
\Delta t_{white} = \frac{0.3\times800}{20} = 12\;{}^\circ\mathrm{C}
3
深色屋面:
\Delta t_{dark} = \frac{0.9\times800}{20} = 36\;{}^\circ\mathrm{C}
✅ 两者相差 24℃!深色屋面相当于外界温度额外高 24℃,漏热按 q=KΔt 成比例增加—— 这就是"冷库刷白"的量化价值。

常见误区

❌
用摄氏度代入 T⁴? → ✔ 必须用开尔文,否则错得离谱。
❌
白色只是好看/卫生? → ✔ 传热上是为了降低太阳短波吸收率。
❌
α_s 和 ε 是一回事? → ✔ 分别对应短波与长波两个波段;理想 cool roof 要 α_s 小、ε 大。
❌
辐射需要空气传播? → ✔ 无需介质,真空也能传——这正是它与导热/对流的本质区别。

与其他知识点的联系

辐射给 KP01 提供屋面边界条件(综合温度 t_{sa});外表面 h_o 实际含 对流+辐射两部分(KP04/KP06);库内冷表面的辐射与结霜凝露耦合(KP07)。晴朗夜空当量温度可比气温低 10~20℃, 高发射率表面夜间还能靠辐射"倒贴"散热。
