物理图景

墙的长宽远大于厚度,热量几乎只沿厚度方向流动(一维);运行足够久后温度不再随时间变化(稳态), 此时热流密度 q 沿厚度处处相等——单层内温度是直线,多层是分段折线,λ 越小的层温降越陡。

核心链条

傅里叶定律 q = −λ·dT/dx → 单层 q = λΔT/δ → 热阻 R = δ/λ(与欧姆定律 I=U/R 完全同构)→ 多层串联 Rtotal = 1/ho + Σδ/λ + 1/hi → 传热系数 K = 1/Rtotal。

典型数值(本页默认参数即此例)

200mm 混凝土(λ=1.5)+ 100mm 聚氨酯(λ=0.025),to=30℃、ti=−18℃、ho=23、hi=8: Rtotal≈4.30,K≈0.23 W/(m²·K),q≈11.2 W/m²;保温层承担 44.7℃ 温差,占 48℃ 总温差的 93%。 冷藏库外墙一般要求 K ≤ 0.3~0.4,冻结库更低。

常见误区

墙内温度是曲线?——稳态、常物性单层内是直线,多层是折线。

K 和 λ 是一回事?——λ 是材料属性;K 是整堵墙含表面换热的综合指标。

保温越厚越好?——存在"经济厚度":运行费与初投资之和最小,且过厚挤占库容。
