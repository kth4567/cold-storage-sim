货物降温是一场两棒接力:第一棒,冷风与货物表面之间对流换热; 第二棒,内部靠导热把热"接力"送到表面。货物外表面正是两段热阻的交接点。 本知识点承上启下:把外部对流(KP04/KP06)与内部非稳态导热(KP09)连接起来, 并引出整个冷却问题的核心判据——毕渥数 Bi。

物理图景

  • 冷风(−18℃)吹过 20℃ 的货物:表面先被冷却,表面与内部产生温差,热量从内部导向表面、再由表面对流给冷风。
  • 两段热阻串联:外部对流热阻 1/h 与内部导热热阻 L_c/λ。 哪段大,哪段就是瓶颈,控制整个降温速率。
  • 表面换热强(h 大)、货物大/导热差 → 内部是瓶颈,中心明显滞后(Bi 大)。
  • 货物小、导热好、换热一般 → 表面是瓶颈,内部几乎同步降温(Bi 小,可用集中参数法)。
一句话记忆:降温 = 表面对流 + 内部导热两段接力;谁的热阻大谁说了算,Bi 就是这个比值。

① 表面能量平衡(交接条件)

h\,(T_s - T_\infty) = -\lambda\,\frac{\partial T}{\partial x}\bigg|_{surface}

任一时刻,对流带走的热 = 内部导来的热——这就是非稳态导热的第三类边界条件。

② 毕渥数 Bi(本知识点的核心)

Bi = \frac{h\,L_c}{\lambda_{goods}} = \frac{L_c/\lambda_{goods}}{1/h} = \frac{\text{内部导热热阻}}{\text{表面对流热阻}}

L_c:特征尺寸(平板取半厚、球取 R/3)。 ⚠ Bi 用货物的 λ;Nu 才用流体的 λ——形似意不同!

③ 判据与分流(整个冷却问题的岔路口)

Bi < 0.1:\;内部近乎均温 \rightarrow 集中参数法(KP10)\qquad Bi > 0.1:\;中心显著滞后 \rightarrow 非稳态导热(KP09)

④ 工程要点

  • 降温瓶颈诊断:先算 Bi。Bi 大 → 加大风速收益有限,关键在减小尺寸/分装;Bi 小 → 提高 h(加风)最有效。
  • 包装的影响:纸箱、泡沫在表面又加一层串联热阻,显著拖慢降温——预冷常要求打孔箱、留通风道。
  • 预冷设计:果蔬"7/8 冷却时间"的本质是让中心尽快达标,由 Bi 与尺寸主导。

降温接力路径 冷风 →(对流膜)→ 表面 →(导热)→ 中心

Bi 热阻跷跷板 左臂 1/h(表面) vs 右臂 L_c/λ(内部)——往哪边倾,瓶颈就在哪边

💡 加大 h,左臂变短、跷跷板右倾——内部成瓶颈,再加风也没用;把货物分装变小,右臂缩短,Bi 掉下来,方法都跟着换。

例题:这箱货能用"简单方法"算吗?

球形货物等效半径 R=0.1m,λ_{goods}=0.5 W/(m·K), 冷风强制对流 h=30 W/(m²·K)。求 Bi,判断能否用集中参数法;若分装成 R=0.02m 的小包装呢? (实验节默认参数即本题)

1
球的特征尺寸:
L_c = \frac{R}{3} = \frac{0.1}{3} = 0.0333\;\mathrm{m}
2
大货的毕渥数:
Bi = \frac{h L_c}{\lambda} = \frac{30\times0.0333}{0.5} = 2.0 \;\gg\; 0.1
不能用集中参数法——中心显著滞后,须按 KP09 非稳态处理。
3
分装成小包装(R=0.02m):
L_c = \frac{0.02}{3} = 0.00667\;\mathrm{m},\qquad Bi = \frac{30\times0.00667}{0.5} = 0.4
仍 >0.1 但已小很多;R=5mm 时 Bi=0.1,恰到临界。
✅ 大货 Bi=2.0 走 KP09;分装后 Bi 大幅下降——减小尺寸比一味加风更能加快中心降温, 这就是"分装预冷"的定量依据。

常见误区

❌
Bi 里的 λ 用空气的? → ✔ Bi 用货物 λ(内部热阻),Nu 才用流体 λ。
❌
加大风速总能加快降温? → ✔ Bi 大时瓶颈在内部,提 h 收益锐减。
❌
忽略包装热阻? → ✔ 包装是表面外又一层串联热阻,明显拖慢降温。
❌
表面温度 = 风温? → ✔ 只有 Bi 很大时才近似成立,一般介于风温与中心温之间。

与其他知识点的联系

h 来自 KP04/KP06(接力第一棒);Bi>0.1 分流去 KP09,Bi<0.1 分流去 KP10; 表面同时是水分蒸发/结露的传质界面——高 h 加快降温也加快干耗,需权衡。
