盘管表面低于 0℃ 又低于空气露点时,库内湿空气中的水蒸气会在翅片上凝华结霜。 霜多孔、含大量空气、导热系数极低——附着在换热面上等于给盘管多穿了一层"棉衣", 直接拉低蒸发器的 U 和换热能力;它还堵塞翅片间隙、减小风量,双重削弱。 理解霜层热阻,就理解了"为什么换热越用越差、为什么要除霜、除霜怎么算账"。

物理图景

  • 湿空气流过冷盘管 → 水蒸气分压高于表面饱和分压 → 水蒸气向表面迁移并凝华(气→固)成霜,放出凝华潜热。
  • 霜层多孔:初期疏松密度低,后期渐致密;λ_{frost} 只有 0.05~0.5 W/(m·K), 远小于金属——对换热是纯坏事。
  • 正反馈:霜越厚 → 换热越差 → 盘管更冷 → 结霜更快;同时翅隙变窄、风量下降。
  • 衰减到一定程度就要除霜(电加热/热气/水冲)——除霜耗能且期间停止制冷,"何时除霜"是一笔经济账。
一句话记忆:霜 = 低导热多孔层,附加热阻 δ/λ 拉低 U;换热锯齿式衰减,需周期除霜。

① 霜层 = 插进 U 里的一层附加串联热阻

\frac{1}{UA} = \frac{1}{\eta_o h_o A_o} + \underbrace{\frac{\delta_{frost}}{\lambda_{frost} A_o}}_{霜层(新增)} + \frac{\ln(r_o/r_i)}{2\pi\lambda L} + \frac{1}{h_i A_i}

霜层热阻 R_{frost} = δ/λ_{frost}, 随霜厚线性增长——KP01 一维导热热阻的直接应用。

② 换热能力衰减

\frac{Q_{frost}}{Q_0} = \frac{R_0}{R_0 + R_{frost}} = \frac{1}{1 + R_{frost}/R_0}

R₀:无霜时空气侧+管壁热阻。霜层热阻相对 R₀ 越大,衰减越狠。

③ 风量下降的叠加效应(更狠的一刀)

\dot V \downarrow \;\Rightarrow\; h_o \propto v^{0.6\sim0.8} \downarrow \;\Rightarrow\; R_0' = R_0\left(\frac{v_0}{v_{eff}}\right)^{0.7} \uparrow

霜堵塞翅隙 → 风量下降 → 空气侧对流也被削弱——导热+对流双重衰减,实际比只算霜层更严重。

④ 结霜的传质本质与除霜能量账

\dot m_{frost} = h_m\,A\,(\rho_{v,\infty} - \rho_{v,s})

空气越湿、盘管越冷、风速越大,结霜越快(刘伊斯关系联系 h 与 h_m)。 除霜要付:融霜潜热 + 加热显热 + 漏进库内的热;除霜频率 = 换热损失账 vs 除霜能耗账的平衡。

例题:2mm 的霜,偷走多少换热能力?

无霜时蒸发器空气侧+管壁等效热阻 R₀=0.04 m²·K/W。 运行后结霜 δ=2mm,霜导热 λ_{frost}=0.15 W/(m·K)。求换热衰减; 若堵塞同时使风量下降令 R₀ 增大 20%,再评估。(实验节的模型即本题)

1
霜层热阻:
R_{frost} = \frac{\delta}{\lambda} = \frac{0.002}{0.15} = 0.0133\;\mathrm{m^2{\cdot}K/W}
2
只算霜层的衰减:
\frac{Q}{Q_0} = \frac{R_0}{R_0 + R_{frost}} = \frac{0.04}{0.0533} = 0.75
3
叠加风量下降(R₀′ = 0.048):
\frac{Q}{Q_{0,原}} = \frac{0.04}{0.048 + 0.0133} = \frac{0.04}{0.0613} \approx 0.65
✅ 单算霜层剩 75%(损失 25%);叠加堵风只剩 65%——堵塞比霜本身更凶, 这就是"按需除霜"要同时盯霜厚和风阻的原因。

常见误区

❌
霜是冰,导热还行? → ✔ 霜多孔含气,λ 很低,是隔热层。
❌
只算导热热阻? → ✔ 霜还堵塞流道降风量,双重衰减。
❌
除霜越勤越好? → ✔ 除霜耗能且把热带进库,有最优频率。
❌
结霜只看温度? → ✔ 结霜是传质过程,还取决于湿度与风速。

与其他知识点的联系

霜层热阻插进 KP05 的 U 分母;堵风降 h_o 接 KP06;湿气来源是 KP04 的开门渗湿; δ/λ 本身就是 KP01 的一维导热热阻。工程上定时除霜简单但易过/欠除霜,按需除霜(风阻/温差/霜厚传感)更节能; 源头治理靠减少开门渗湿、翅片间距设计、尽量提高蒸发温度。

KP01 导热热阻KP05 换热器 KP06 强制对流KP04 开门渗湿
要点小结:R_frost = δ/λ 随霜厚线性涨;换热比 = R₀/(R₀+R_frost),叠加堵风更狠; 除霜是"损失账 vs 能耗账"的平衡——锯齿曲线下的平均能效才是真指标。
