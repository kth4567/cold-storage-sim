# 数字冷库 · 传热学知识点文档包 · 总索引

> 本文档包基于《数字冷库传热学知识点映射表》生成，服务于"数字冷库"交互式教学网页项目：用户点击冷库的某个部位，浮窗弹出对应的传热学知识点，并可进入该知识点的专属交互 HTML 页面深入学习。
>
> **每个知识点 = 两份 markdown**：
> 1. `KPxx_知识点讲解.md`——非常详细的物理知识讲解（定位/图景/公式/工程应用/例题/误区/联系/小结）。
> 2. `KPxx_HTML技术路线.md`——该知识点专属 HTML 页面**如何设计**（为何这样呈现/核心交互隐喻/布局/组件/计算模型/技术选型/衔接/里程碑/验收）。**每一页的隐喻互不雷同**，与该知识点的物理本质精准契合。

## 二、7 大知识类别归类（对应映射表 sheet2）

1. **导热**：KP01（平壁一维稳态）、KP11（圆筒壁，含管内对流）
2. **非稳态导热**：KP03（半无限大）、KP09（一般非稳态·中心滞后）、KP10（集中参数·Bi<0.1 极限）
3. **对流传热**：KP04（自然对流·门板）、KP06（强制对流·风机）、KP08（表面传热·引出 Bi）
4. **相变传热**：KP05（换热器·制冷剂蒸发）、KP07（霜层·结霜相变）
5. **辐射传热**：KP02（屋面·太阳辐射）
6. **传质**（结霜含水汽传质，蕴含于 KP07）
7. **热负荷分析**：贯穿各页的冷损失/放热量汇总（KP01/05/07/09/10/11 均回填）

## 四、教学线索（推荐串讲顺序）

1. **围护结构得热**（稳态起步）：KP01 墙体 → KP02 屋面 → KP11 管道（导热三形态：平壁/曲面场景），穿插 KP03 地板（非稳态入门）。
2. **对流三兄弟**：KP04 自然对流 → KP06 强制对流 → KP08 表面传热（用 Bi 承接内外）。
3. **货物降温主线**（全包最重要一条）：KP08 Bi 判据 →（分流）→ KP09 非稳态（Bi>0.1，中心滞后）/ KP10 集中参数（Bi<0.1，整块均匀）。**KP08→KP09/KP10 是核心分叉**。
4. **蒸发器专题**：KP05 换热器 → KP06 风机强制对流 → KP07 霜层热阻 → KP11 盘管管道，构成"蒸发器换热全景"。
5. **收束**：各页冷损失/放热量回填**热负荷分析**，形成闭环。
