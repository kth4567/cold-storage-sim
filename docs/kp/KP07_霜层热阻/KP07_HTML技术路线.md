# KP07 · HTML 技术路线：霜层热阻

> 说明"霜层热阻"专属 HTML 页面**如何设计**，非代码交付。

---

## 1. 为什么这个知识点适合这样呈现？

霜层热阻的独特叙事是**"时间推移中霜越积越厚 → 一根热阻条越叠越高 → 换热能力表针越掉越低 → 一键除霜清零 → 循环往复"**。它天生是一个**"随时间恶化、可被除霜重置"的锯齿循环**，这是别的知识点都没有的时间形态：
1. **霜在长厚** → 主视觉是**盘管翅片上逐渐增厚的霜层**（可见地长出来）。
2. **热阻叠加** → 天然对应一根**从底部长高的"霜层热阻"段**，叠在原有热阻之上。
3. **换热衰减** → 天然对应一个**换热能力/制冷量表针**随时间下滑。
4. **除霜重置** → 天然对应一个**除霜按钮**，按下霜化掉、表针回升、能耗计数跳一格，形成锯齿曲线。

因此本页核心是**"结霜生长动画 + 热阻堆叠条 + 换热能力衰减曲线 + 除霜按钮的锯齿循环"**——一个**"时间序列 + 运维决策"**页面。

---

## 2. 核心交互隐喻

**"看着霜慢慢长厚、制冷量一路下滑；什么时候按除霜？按早了浪费能耗，按晚了换热太差——你来权衡。"** 这是本页独有记忆点：**一个除霜按钮，一条锯齿能效曲线。**

## 4. 关键交互组件与实现逻辑

### 4.1 结霜生长动画（主视图，本页灵魂）
- **Canvas/SVG**：盘管翅片上覆一层霜厚 $\delta_{frost}(t)$，随时间按结霜速率增厚；翅片间隙相应变窄，可用颜色/纹理表现霜由疏松→致密。
- 结霜速率由湿度、蒸发温度、风速决定（传质），滑杆调节。

### 4.2 热阻堆叠条（把"附加热阻"演出来）
- 竖向分段条：管壁 R、空气侧 R、**霜层 R（动态增高）**。
- 霜层段高度 ∝ $\delta_{frost}/\lambda_{frost}$，实时增长，直观显示"多穿一层棉衣"。

### 4.3 换热能力衰减曲线（锯齿）
- **SVG** 折线：$Q/Q_0=1/(1+R_{frost}/R_0)$ 随时间下滑；风量下降令 $R_0$ 增大，衰减更陡。
- 除霜时曲线**垂直跳回**100%，形成锯齿；累计能耗（结霜期损失 + 除霜能耗）实时统计。

### 4.4 除霜按钮与最优决策
- "除霜"按钮：按下 → 霜厚清零、曲线回弹、除霜能耗计数 +1、短暂"停机漏热"提示。
- 提供"自动最优除霜"模式：系统在净能效最优时刻自动除霜，与用户手动对比，教学"何时除霜"。

## 5. 页面实时计算模型

```
// 结霜速率(传质，示意)
dδ/dt ∝ h_m * (ρv_air - ρv_surface)     // 随湿度、风速、盘管温度
R_frost = δ_frost / λ_frost
// 风量随堵塞下降 → 空气侧热阻上升
v_eff = v0 * (1 - k*δ_frost)
R0_eff = R0_base * (v0/v_eff)^0.7
Q_ratio = R0_eff_noFrost / (R0_eff + R_frost)   // 相对原始能力
// 能耗账
E_run_loss += (1 - Q_ratio)*dt                  // 换热不足的代价
E_defrost  += E_per_defrost (每次除霜)
```

锯齿曲线下的"平均能效"是评价除霜频率的目标函数。

---

## 6. 可视化技术选型（为何这样选）

- **Canvas/SVG（结霜 + 翅片）**：霜层增厚是渐变形状动画，SVG（形状可控）或 Canvas（纹理细腻）均可；霜的多孔纹理用 Canvas 更传神。
- **SVG（热阻堆叠条 + 锯齿曲线）**：需要精确高度比例与时间轴曲线，SVG 清晰。
- 本页的技术核心不是流场而是**时间序列 + 状态重置**，与前几页都不同——形态直接呼应"结霜-除霜循环"。

## 7. 与主"数字冷库"页面的衔接

- 入口：主场景点击**蒸发器霜层**热点 → 浮窗"霜层热阻……" → "深入了解"进入本页。
- 联动：霜层 R 回传 KP05 抬高 U 分母、风量下降回传 KP06 降低 $h_o$；结霜速率与库内湿度联动 KP04 开门渗湿——本页是"多知识点后果的汇聚"。
- 身份标识：**结霜生长 + 热阻堆叠 + 除霜锯齿曲线**是本页独有形态。

---

## 8. 技术栈建议

- Canvas（霜/翅片）+ SVG（热阻条、锯齿曲线）+ HTML 控制/计数。
- 时间引擎 `requestAnimationFrame` 驱动 δ_frost 增长与曲线滚动；除霜为状态重置事件。
- 纯前端，无后端。

---

## 9. 实现里程碑

1. 静态盘管 + 热阻堆叠条 + Q_ratio 计算（对齐例题 25% 损失）。
2. 加时间引擎，霜层增厚、热阻条长高、Q 下滑。
3. 加锯齿曲线与除霜按钮（手动清零 + 能耗计数）。
4. 加风量下降耦合（衰减更陡，对齐例题 65%）。
5. 加自动最优除霜模式与手动对比。

---

## 10. 验收标准

- 随时间霜可见增厚、翅隙变窄、热阻条增高、换热能力表针下滑；数值与 KP07 例题一致（单霜 75%、叠堵塞 65%）。
- 除霜按下曲线垂直回弹形成锯齿，能耗计数累加。
- 自动最优除霜的平均能效优于随意手动除霜。
- 页面形态（**结霜生长 + 除霜锯齿循环**）**只可能属于"霜层热阻"**——达成精准契合。
