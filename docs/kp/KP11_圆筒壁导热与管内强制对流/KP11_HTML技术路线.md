# KP11 · HTML 技术路线：圆筒壁导热 + 管内强制对流

> 说明"圆筒壁导热 + 管内强制对流"专属 HTML 页面**如何设计**，非代码交付。

---

## 1. 为什么这个知识点适合这样呈现？

本知识点的灵魂是**"径向"**：一切都围绕一根管子的**横截面**由内向外发生——制冷剂在圆心流动，热量沿**半径**一层层往外穿（管内对流膜 → 铜管壁 → 保温层 → 外部空气）。这与前面所有页面的几何都不同：KP01 是**横向平壁分层**、KP09 是**时间轴演化**，而本页必须是**同心圆的、从圆心向外辐射**的视角。

因此本页唯一贴切的主视觉是**"管道横截面的同心圆环 + 沿半径的温度分布曲线"**：

1. **同心圆环剖面** → 管内制冷剂（圆心）、铜管壁、保温层、外部空气，一圈套一圈，直接映射四段串联热阻。
2. **沿半径的温度曲线** → 横轴是半径 r（从 $r_1$ 到 $r_3$），温度沿半径下降，且因为是对数律，**曲线在保温层段是弯的**（不像平壁的直线）——这是圆筒导热的视觉签名。
3. **临界半径**这一圆管独有现象 → 天然对应一个"**总散热 vs 保温外径**"的**先升后降/单调曲线**，可拖动保温厚度看散热怎么变。
4. **穿墙冷桥** → 一个可切到"纵剖 + 穿墙"的视图，冷桥处**发光泄漏**。

所以本页核心是**"同心圆环剖面 + 沿半径对数温度曲线 + 临界半径寻优 + 穿墙冷桥发光"**——一个**径向/极坐标视角**的页面，是全套里唯一"从圆心向外看"的一页。

---

## 2. 核心交互隐喻

**"低头看一根管子的横截面：制冷剂在圆心，冷量一圈圈往外穿，每穿一层就爬一段温度台阶；你能拖动保温层厚度，看沿半径的温度曲线怎么弯、看总冷损失曲线上那个'临界半径'的拐点；再切到穿墙视图，会看到没包好的地方正发着光往外漏冷。"** 这个"同心圆环 + 径向温度台阶 + 临界半径拐点"是本页独有记忆点。

## 4. 关键交互组件与实现逻辑

### 4.1 同心圆环剖面（左上，SVG/Canvas，本页招牌）
- 四个同心区：制冷剂芯、铜管壁（$r_1\!\to\!r_2$）、保温层（$r_2\!\to\!r_3$）、外部空气。
- 半径可**按真实比例或对数缩放**切换（保温远厚于管壁，需缩放才能都看清）。
- 颜色按各半径处温度着色，形成"由内到外"的冷—暖过渡；鼠标滑到某半径显示该处温度。
- 拖动 $r_3$（保温厚度）时圆环外圈实时变粗/变细。

### 4.2 沿半径温度分布曲线（右上，SVG）
- 横轴 r（$r_1$ 到 $r_3$，可对数轴）、纵轴 T。
- 关键：**管内/外对流是"温度台阶"、圆筒导热段是对数曲线（弯的）**——务必如实画出弯曲，与 KP01 平壁的折线形成对照。
- 各分界半径 $r_1,r_2,r_3$ 处标注温度；与左侧圆环的半径游标联动。

### 4.3 四段热阻串联条（左下）
- 横向堆叠条：管内对流 / 管壁 / 保温 / 外对流，长度∝各段热阻，实时显示占比（例题里保温占 91%）。
- 与 KP05 的"串联热阻条"呼应但为圆筒式，点明"谁是主阻"。

### 4.4 临界半径寻优（右下，本页独有现象）
- 曲线：横轴保温外径 $r_3$、纵轴总散热 Q（或 $q_l$）。
- 标出 $r_{cr}=\lambda_{ins}/h_o$ 的位置；当前 $r_3$ 用游标落在曲线上。
- 拖动 λ_ins 或 h_o 改变 $r_{cr}$，让用户看到"$r_2<r_{cr}$ 时加保温反增散热"的反直觉现象。这是圆管相对平壁最独特的知识点，必须做足。

### 4.5 视图切换：穿墙冷桥（纵剖）
- 切到"穿墙纵剖"：管子横穿一堵库墙，保温**连续**时冷桥处正常；切到"保温断开"时，断桥处**发光 + 冷量粒子外漏 + 局部结露高亮**（接 KP07 霜层）。
- 用一个开关对比"连续保温 vs 断桥"的冷损失差，直观教育穿墙节点处理。

### 4.6 管内对流旋钮（Re → h_i）
- 调流速/Re，用 Dittus-Boelter 实时算 $h_i$，管内对流台阶高度随之变化，呼应 KP06。

## 5. 实时计算模型（前端可跑）

- 管内对流：$\mathrm{Re}=\rho u d/\mu$，$\mathrm{Nu}=0.023\mathrm{Re}^{0.8}\mathrm{Pr}^n$，$h_i=\mathrm{Nu}\,\lambda_f/d$。
- 四段热阻（单位长）：$R_1=\frac1{h_i2\pi r_1}$，$R_2=\frac{\ln(r_2/r_1)}{2\pi\lambda_{wall}}$，$R_3=\frac{\ln(r_3/r_2)}{2\pi\lambda_{ins}}$，$R_4=\frac1{h_o2\pi r_3}$。
- 冷损失：$q_l=\Delta T/\sum R$；沿半径温度由各段热阻按比例分配（对数段用 $\ln$ 插值）。
- 临界半径：$r_{cr}=\lambda_{ins}/h_o$；扫 $r_3$ 得 $Q(r_3)$ 曲线找拐点。
- 全为封闭式，**浏览器实时**，滑杆流畅联动。

## 6. 可视化技术选型（为何这样选）

- **同心圆环**用 SVG（矢量圆环、易着色与交互）或 Canvas（若要细腻的径向渐变/粒子）——本页需要"极坐标/径向"渲染，是唯一采用同心圆布局的页面。
- **沿半径曲线**用 SVG，重点在如实呈现**对数弯曲**（区别平壁直线）。
- **穿墙冷桥泄漏**用 Canvas 粒子（发光外漏）表现能量损失，情感冲击强。
- 极坐标/径向的整体视角，与 KP01（横向平壁）、KP05（流程双曲线）、KP09（时间轴）在几何构图上均不撞车。

## 7. 与主站及相邻知识点的衔接

- **对照入口**：从 **KP01** 可跳来对比"平壁 δ/λ ↔ 圆筒 ln(r₂/r₁)"；管内对流台阶链接 **KP06**；串联热阻思路链接 **KP05**。

- 作为"集大成"页，可在主站作为**收束节点**，回链多条知识线。

## 8. 技术栈与里程碑

- **技术栈**：SVG（同心圆环 + 半径温度曲线 + 热阻条 + 寻优曲线）+ Canvas（穿墙冷桥粒子）+ JS 计算层。
- **里程碑**：
  1. 同心圆环剖面 + 沿半径对数温度曲线（含台阶与弯曲）；
  2. 四段热阻串联条 + 冷损失 $q_l$ 实时；
  3. 临界半径寻优曲线（拖 λ_ins/h_o 看拐点）；
  4. 穿墙纵剖冷桥视图（连续 vs 断桥对比、发光泄漏）+ 管内 Re→h_i 旋钮。

## 9. 验收标准

- 圆环剖面**从圆心向外**分四层，配色与沿半径温度一致；温度曲线在保温段呈**对数弯曲**（非直线）。
- 拖保温厚度 $r_3$，冷损失 $q_l$ 与临界半径曲线联动正确；能复现"$r_2<r_{cr}$ 加保温反增散热"。
- 穿墙视图能对比连续保温 vs 断桥，断桥处发光泄漏 + 结露高亮。
- 采用**同心圆/径向**构图，与其余 10 页几何视角均不重复，作为"集大成"页正确回链 KP01/05/06/07。
