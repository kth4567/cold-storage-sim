# KP11 圆筒壁导热 + 管内强制对流（风机内蒸发器管道）

> **冷库部位**：蒸发器管道 / 穿墙制冷剂管路
> **浮窗标题**：圆筒壁导热 + 管内强制对流
> **浮窗简短说明**：制冷剂在圆管内强制流动换热，热量再径向穿过管壁与保温层，构成串联热阻；穿墙处易成冷桥。
> **知识类别**：导热（圆筒坐标）+ 对流（管内强制对流）

---

## 1. 知识点定位

前面的墙体（KP01）用的是**平壁**导热模型，但冷库里大量的换热发生在**圆管**里：蒸发器盘管、制冷剂供液/回气管、穿墙管路。圆管的几何决定了导热必须用**圆筒坐标**，且传热面积**沿半径变化**——这带来平壁没有的新现象（如**临界保温半径**）。同时，管内制冷剂是**强制流动**的，需要**管内强制对流**（Dittus-Boelter 类关联式）来定 h。

本知识点是全套的"**集大成者**"：它把**管内强制对流（KP06 的圆管版）+ 圆筒壁径向导热（KP01 的曲面版）+ 多层串联热阻（KP05 的思路）**缝合在一根真实的管子上，并落到冷库最痛的工程点——**穿墙冷桥/冷损失**。它是"从平面到曲面、从单壁到多层管"的升级站，也是把前面多条线索收束的终点。

---

## 2. 物理图景

- 制冷剂在管内**强制流动**，与管内壁对流换热（第一段热阻，$1/(h_i A_i)$）。
- 热量再**径向**穿过金属管壁（第二段，圆筒导热）、穿过外包保温层（第三段，圆筒导热）。
- 最后从保温外表面对流/散给周围（第四段，$1/(h_o A_o)$）。
- 四段**串联**，热流沿半径向外一路"闯关"。与平壁不同：**面积随半径增大**，所以外层单位厚度的"导热阻力"没有内层大——这就埋下了**临界保温半径**的伏笔。
- **穿墙处**：管子穿过库墙时，若保温不连续，就形成一条低热阻的"**冷桥**"，冷量顺着管子大量漏出/热量漏入，局部结露结霜。

一句话：**热量在管里先被制冷剂"对流接走"，再径向穿过管壁与保温层层层串联；面积随半径变化带来临界半径，穿墙不连续则成冷桥。**

---

## 3. 核心概念与公式

### 3.1 圆筒壁径向稳态导热（单层）

对内径 $r_1$、外径 $r_2$、长 $L$、导热系数 $\lambda$ 的圆筒壁：

$$Q = \frac{2\pi\lambda L\,(T_1 - T_2)}{\ln(r_2/r_1)}$$

对应**单层圆筒导热热阻**：

$$R_{cyl} = \frac{\ln(r_2/r_1)}{2\pi\lambda L}$$

> 与平壁对照：平壁 $R=\delta/(\lambda A)$（线性厚度）；圆筒 $R=\ln(r_2/r_1)/(2\pi\lambda L)$（**对数**半径比）。这是"曲面导热"的标志性差别。

### 3.2 管内强制对流（定 $h_i$）

制冷剂/空气在管内湍流时，用 **Dittus–Boelter 关联式**：

$$\mathrm{Nu} = 0.023\,\mathrm{Re}^{0.8}\,\mathrm{Pr}^{n}\quad(n=0.4\text{ 加热}/0.3\text{ 冷却})$$

$$\mathrm{Re}=\frac{\rho u d}{\mu},\qquad h_i = \frac{\mathrm{Nu}\,\lambda_{fluid}}{d}$$

管内换热强度随流速（Re）显著上升——这与 KP06 的强制对流同源，只是几何换成"管内"、特征尺寸取管径 d。

### 3.3 多层串联总热阻与传热量

从管内制冷剂到管外环境，四段串联：

$$R_{tot} = \underbrace{\frac{1}{h_i\,2\pi r_1 L}}_{\text{管内对流}} + \underbrace{\frac{\ln(r_2/r_1)}{2\pi\lambda_{wall}L}}_{\text{管壁导热}} + \underbrace{\frac{\ln(r_3/r_2)}{2\pi\lambda_{ins}L}}_{\text{保温导热}} + \underbrace{\frac{1}{h_o\,2\pi r_3 L}}_{\text{外部对流}}$$

$$Q = \frac{T_{\infty,i}-T_{\infty,o}}{R_{tot}}$$

工程上常用**单位管长热流** $q_l = Q/L$（W/m）比较。这与 KP05 的"串联热阻链"思路完全一致，只是各段换成圆筒表达式。

### 3.4 临界保温半径（圆管独有现象）

加保温会**增加导热热阻**（$\ln r_3/r_2$ 项）、但同时**增大外表面积**（$2\pi r_3 L$）从而**减小外对流热阻**。两者相反，存在一个使总散热最大的**临界半径**：

$$r_{cr} = \frac{\lambda_{ins}}{h_o}$$

- 当 $r_2 < r_{cr}$：初期加保温**反而增大散热**（小管、细线常见）。
- 冷库保温管 $r_2$ 通常已远大于 $r_{cr}$，故加保温是**有效减损**的——但小径管、临界情形须校核，避免"越包越漏"。

### 3.5 穿墙冷桥

穿墙管若保温断开或金属直接接触墙体，形成局部低热阻通道：

$$Q_{bridge}\ \uparrow\quad(\text{局部 }R\ \downarrow)$$

后果：冷量损失、穿墙内壁**结露/结霜**（表面降到露点以下，见 KP07 霜层）。对策：保温连续包裹、加隔热套管、断桥处理。

## 4. 冷库工程应用

1. **蒸发器盘管选型**：管内强制对流 $h_i$ 决定制冷剂侧换热强度；配合翅片、管排（KP05 换热器）确定盘管 UA。
2. **管路保温设计**：用总串联热阻算单位管长冷损失 $q_l$，据此选保温厚度；对小径管须校核**临界半径**，避免无效甚至反效保温。
3. **穿墙/穿板节点**：穿墙管是冷桥高发区，须连续保温 + 断桥，防止冷损失与结露结霜——直接关系围护结构寿命与能耗。
4. **供液/回气管平衡**：不同管段温度、流速不同，$h_i$、$q_l$ 各异，用本模型逐段核算，指导管径与保温分级。

## 5. 典型例题（带数值）

**题目**：一根蒸发器供液铜管，内半径 $r_1=8\,\text{mm}$、外半径 $r_2=10\,\text{mm}$，外包保温层到 $r_3=40\,\text{mm}$。铜 $\lambda_{wall}=380$、保温 $\lambda_{ins}=0.04\,\text{W/(mK)}$。管内制冷剂对流 $h_i=1500$、保温外表面 $h_o=8\,\text{W/(m}^2\text{K)}$。制冷剂 $-30\,℃$、库外空气 $+5\,℃$。求每米管长的冷损失 $q_l$，并判断保温是否有效。

**解**（取单位长 $L=1\,\text{m}$）：

1. 管内对流热阻：
$$R_1=\frac{1}{h_i\,2\pi r_1}=\frac{1}{1500\times2\pi\times0.008}\approx\frac{1}{75.4}\approx0.0133\ \text{K·m/W}$$

2. 铜管壁导热（极小）：
$$R_2=\frac{\ln(r_2/r_1)}{2\pi\lambda_{wall}}=\frac{\ln(10/8)}{2\pi\times380}=\frac{0.223}{2388}\approx9.3\times10^{-5}\ \text{（可忽略）}$$

3. 保温层导热（主阻）：
$$R_3=\frac{\ln(r_3/r_2)}{2\pi\lambda_{ins}}=\frac{\ln(40/10)}{2\pi\times0.04}=\frac{1.386}{0.251}\approx5.52\ \text{K·m/W}$$

4. 外部对流：
$$R_4=\frac{1}{h_o\,2\pi r_3}=\frac{1}{8\times2\pi\times0.040}=\frac{1}{2.01}\approx0.497\ \text{K·m/W}$$

5. 总热阻与冷损失：
$$R_{tot}=0.0133+0.0001+5.52+0.497\approx6.03\ \text{K·m/W}$$
$$q_l=\frac{\Delta T}{R_{tot}}=\frac{5-(-30)}{6.03}=\frac{35}{6.03}\approx5.8\ \text{W/m}$$

6. 临界半径校核：
$$r_{cr}=\frac{\lambda_{ins}}{h_o}=\frac{0.04}{8}=0.005\,\text{m}=5\,\text{mm}$$
因 $r_2=10\,\text{mm} > r_{cr}=5\,\text{mm}$，**加保温确实减少散热**（有效）。保温层贡献了总热阻的 $5.52/6.03\approx91\%$，是绝对主力。

**结论**：每米冷损失约 **5.8 W**；保温有效且是主阻。若把保温减薄使 $r_3$ 接近 $r_{cr}$，冷损失会显著上升——量化了"保温该包多厚"。

## 6. 常见误区

- **误区一：用平壁公式算圆管。** 圆管导热是 $\ln(r_2/r_1)$ 而非 $\delta/\lambda$；管壁薄时两者接近，但保温厚时差别很大，必须用圆筒式。
- **误区二：以为加保温一定减少散热。** 存在**临界半径** $r_{cr}=\lambda_{ins}/h_o$；当 $r_2<r_{cr}$，初期加保温反而增大散热（小管/细线要警惕）。
- **误区三：忽略管内对流热阻。** 低流速或大管径时 $1/(h_iA_i)$ 不可忽略；须用 Dittus-Boelter 先定 $h_i$。
- **误区四：漏掉穿墙冷桥。** 穿墙节点是冷损失与结露的高发点，保温必须连续、断桥处理。
- **误区五：Nu 里 n 取错。** 加热 n=0.4、冷却 n=0.3；蒸发器管内制冷剂被"加热"蒸发，注意工况。

- **← KP01 一维稳态导热（平壁）**：本页是"曲面/圆筒版"，核心对照 $\delta/\lambda$ ↔ $\ln(r_2/r_1)/2\pi\lambda L$，并新增临界半径这一圆管独有现象。

- **↔ 热负荷分析**：单位管长冷损失 $q_l$ 汇总为管路系统的冷损失负荷。

|---|---|
| 圆筒导热 | $Q=\dfrac{2\pi\lambda L\Delta T}{\ln(r_2/r_1)}$，热阻 $\ln(r_2/r_1)/(2\pi\lambda L)$ |

| 与平壁差别 | 面积随半径变化 → 对数关系 + 临界半径 |
| 工程量 | 单位管长冷损失 $q_l=\Delta T/R_{tot}$（W/m） |
