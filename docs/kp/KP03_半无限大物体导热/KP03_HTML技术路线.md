# KP03 · HTML 技术路线：半无限大物体导热

> 说明"半无限大物体导热"专属 HTML 页面**如何设计**的技术路线，非代码交付。

---

## 1. 为什么这个知识点适合这样呈现？

半无限大导热的独特之处在于**"深度 × 时间"两个维度耦合、且冷从上往下缓慢渗透**。它天然要求一个与前面页面都不同的形态：
1. **方向是"向下、向深处"** → 页面主视觉应是一个**竖直向下延伸的土壤剖面**（地坪在顶、深处在底），与 KP01 的横向墙体、KP02 的天空场景都不同。
2. **随时间加深** → 天然对应一个**时间轴/播放器**，拖动或播放时间，看冷"锋面"一点点往下爬。
3. **温度是 (x, t) 的场** → 天然对应一张**深度—时间热力图（heatmap）**，把两个维度一次性铺满。

因此本页核心是**"竖直土壤剖面的温度渗透动画 + 深度-时间热力图 + erf/η 曲线"**：一个以**时间为主角**的非稳态可视化。

---

## 2. 核心交互隐喻

**"拖时间轴，看冷锋一天天往地下爬；爬的速度越来越慢（∝√t），深处始终滞后。"** 这是本页唯一记忆点。

---

## 3. 页面布局与视觉结构

```
┌───────────────────────────────────────────────┐
│  标题：半无限大物体导热 · 冷量向地下渗透          │
├──────────────────────┬────────────────────────┤
│  ▬▬▬ 地坪表面 -18℃ ▬▬  │  深度-时间热力图            │
│  ┃(蓝)             ┃  │   t →                    │
│  ┃  温度色柱        ┃  │  x  ▓▒░ (等温线随√t展开)   │
│  ┃  (上冷下暖)      ┃  │  ↓                       │
│  ┃  渗透锋面 ~4√αt  ┃  │                          │
│  ┃(暖)  深处15℃     ┃  │  erf(η) 曲线：η=x/2√αt    │
│  ▼ 深处（无限）      │                          │
├──────────────────────┴────────────────────────┤
│  时间轴 ◀━━━●━━━▶ 第 30 天   [播放]              │
│  控制：α(土壤湿度) · 表面温度 · 显示渗透深度/表面热流 │
└───────────────────────────────────────────────┘
```

**视觉语言**：竖直剖面强调"向下"；用色柱（顶蓝→底暖）表示温度随深度回升；一条明显的**"渗透锋面线"**（δ_p≈4√αt）随时间下移，是全页的动态焦点。

---

## 4. 关键交互组件与实现逻辑

### 4.1 竖直土壤剖面（主视图）
- **Canvas**：纵向像素条，每个深度像素的颜色 = 该深度当前温度经色标映射。
- 每帧按当前 t，用 $T(x)=T_s+(T_i-T_s)\mathrm{erf}(x/2\sqrt{\alpha t})$ 逐像素填色。
- 叠加"渗透锋面"横线（δ_p）与"0 ℃ 冰线"（防冻关注深度）。

### 4.2 深度—时间热力图（副视图，本页特色）
- 横轴 t、纵轴 x 的二维热力图，一次展示整个温度场演化；一条竖直游标随时间轴移动，与主剖面同步。
- 等温线（如 0 ℃、−10 ℃）呈现为随 √t 张开的抛物线族——直观显示"越往后渗得越慢"。

### 4.3 erf / η 曲线（物理内核）
- **SVG** 画 $\mathrm{erf}(\eta)$ 曲线，标出当前 (x, t) 对应的 η 工作点。
- 让用户理解：不同的 (x, t) 只要 η 相同，无量纲温度就相同（相似解思想）。

### 4.4 时间轴播放器 + 参数
- 时间轴可拖动/播放/调速；α 由"土壤湿度"滑杆决定（湿土 α 大→渗得快），表面温度可调。
- 实时数字：当前渗透深度 δ_p、表面热流 $q_s=\lambda(T_i-T_s)/\sqrt{\pi\alpha t}$（显示其 1/√t 衰减）。

---

## 5. 页面实时计算模型

```
η(x,t) = x / (2*sqrt(α*t))
T(x,t) = T_s + (T_i - T_s) * erf(η)      // erf 用数值近似(Abramowitz-Stegun)
δ_p(t) = 4 * sqrt(α*t)                    // 渗透深度
q_s(t) = λ*(T_i - T_s) / sqrt(π*α*t)      // 表面热流 ∝ 1/√t
```

`erf` 无内置函数，用有理近似实现（误差<1e-7）。逐帧对可见深度像素求值即可。

---

## 6. 可视化技术选型（为何这样选）

- **Canvas（剖面 + 热力图）**：本质是逐像素填色的温度场，像素级绘制正是 Canvas 所长；热力图数据量大，Canvas 性能优于 SVG。
- **SVG（erf 曲线）**：少量曲线 + 工作点标注，SVG 更清晰可交互。
- 与 KP01（SVG 剖面）区别开：这里是**连续温度场**而非几层离散块，故选 Canvas——技术选型本身也"贴合"了物理（连续介质 vs 分层结构）。

---

## 7. 与主"数字冷库"页面的衔接

- 入口：主场景点击**地坪**热点 → 浮窗"半无限大物体导热……" → "深入了解"进入本页。
- 教学联动：本页可显示"0 ℃ 冰线到达深度"，与地坪防冻设计联动；把渗透稳定后的深层等效地温传回 KP01 作为地坪 K 值计算的边界。
- 身份标识：**竖直向下的渗透动画 + 深度-时间热力图**是本页独有形态。

---

## 8. 技术栈建议

- Canvas 2D + 少量 SVG；erf 近似函数自带；播放用 `requestAnimationFrame`。
- 大量像素运算可按需降采样或用 `OffscreenCanvas`/`ImageData` 提速。
- 纯前端，无后端。

---

## 9. 实现里程碑

1. 实现 erf 近似 + 单帧剖面填色（固定 t）。
2. 接时间轴，剖面随 t 演化，渗透锋面下移。
3. 加深度-时间热力图与同步游标。
4. 加 erf/η 曲线与工作点。
5. 加 α（湿度）/表面温度控制、表面热流与冰线读数。

---

## 10. 验收标准

- 播放时间，冷锋向下推进且**明显减速**（体现 √t），深处存在可见滞后。
- 数值与 KP03 例题一致（30 天、1 m 处≈−2.6 ℃，δ_p≈4.5 m）。
- 表面热流随时间按 1/√t 衰减可被读数/曲线证实。
- 页面形态（**竖直渗透 + 时间主导**）**只可能属于"半无限大非稳态导热"**——达成精准契合。
