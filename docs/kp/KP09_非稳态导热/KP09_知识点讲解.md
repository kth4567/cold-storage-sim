# KP09 非稳态导热（货物内部 / 货物中心）

> **冷库部位**：货物内部 / 货物中心
> **浮窗标题**：非稳态导热（中心温度滞后）
> **浮窗简短说明**：货物内部温度随时间变化，中心降温明显滞后于表面。
> **知识类别**：非稳态导热

---

## 1. 知识点定位

前一站（KP08）我们在货物**表面**把"外部对流 + 内部导热"接上了头，并用毕渥数 Bi 判断瓶颈在内还是在外。当 **Bi > 0.1**（内部导热不可忽略）时，货物内部各点温度**不再均匀**、而且**随时间不断变化**——这正是**非稳态导热**（又称瞬态导热、不稳定导热）要处理的问题。

本知识点是整个冷库"货物降温"链条的**心脏**：它回答工程上最关心的那个问题——**"这批货物中心要多久才能降到目标温度？"** 冷库设计的冻结时间、冷却间停留时间、货物码放厚度，全都由非稳态导热决定。它上承 KP08（Bi 判据）、与 KP10（集中参数法是 Bi→0 的极限）互为对照，是"内部路线"里最完整、最核心的一站。

---

## 2. 物理图景

- 一块 20 ℃ 的货物突然被放进 −18 ℃ 冷风中。表面**立刻**开始降温，但热量要从中心"走"到表面需要时间，于是形成**由表及里、逐层滞后**的温度波。
- **早期**：只有表层被"惊动"，中心还蒙在鼓里（类似 KP03 半无限大物体的早期行为）。
- **中后期**：温度扰动传到中心，整块货物进入"规则冷却阶段"，各点温度按同一指数节奏一起下降，**中心始终是最热点**、最后一个达标。
- 温度分布像一条**不断下沉、且中间鼓起**的曲线：表面贴近风温、中心高高凸起，凸起随时间被逐渐"压平并整体下移"。

一句话：**非稳态导热=一场由表及里的降温接力赛，中心永远最后撞线；我们要算的就是撞线时间。**

---

## 3. 核心概念与公式

### 3.1 控制方程（非稳态导热方程 / 傅里叶第二定律）

$$\frac{\partial T}{\partial t} = \alpha \nabla^2 T,\qquad \alpha = \frac{\lambda}{\rho c_p}$$

一维（平板）形式：

$$\frac{\partial T}{\partial t} = \alpha\,\frac{\partial^2 T}{\partial x^2}$$

- $\alpha$：**热扩散率**（m²/s），衡量"温度扰动传播的快慢"，是本知识点第一主角。$\alpha$ 大 → 内部拉平快、中心跟得紧。
- 它与稳态导热（KP01）的根本区别：等号左边多了随时间变化的 $\partial T/\partial t$，说明**物体在蓄热/放热**，不再是"进=出"。

### 3.2 两个决定性无量纲数：Bi 与 Fo

非稳态导热的解可以整理成只依赖两个无量纲数的形式：

$$\text{Bi} = \frac{h L_c}{\lambda_{goods}}\quad(\text{空间：内外热阻比，见 KP08}),\qquad
\text{Fo} = \frac{\alpha t}{L_c^{2}}\quad(\text{时间：无量纲时间})$$

- **傅里叶数 Fo** 是本知识点核心：它是"**无量纲时间**"，表示温度扰动已经渗透了多深/多充分。Fo 越大，说明降温进行得越"透"。
- **Bi 管形状、Fo 管进度**：给定 Bi，中心的无量纲温度只是 Fo 的函数。

### 3.3 无量纲温度与规则阶段（一项近似）

定义无量纲过余温度：

$$\theta^* = \frac{T - T_\infty}{T_i - T_\infty}\in[0,1]$$

当 $\text{Fo} > 0.2$（度过初始阶段后），级数解只保留第一项，**中心**温度可写成简单指数形式：

$$\theta^*_{center} = C_1\,\exp\!\big(-\zeta_1^{2}\,\text{Fo}\big)$$

其中 $C_1,\ \zeta_1$ 是只与 **Bi** 和几何形状（平板/圆柱/球）有关的常数（查表得到）。这说明**规则阶段里降温是"指数衰减"**——与 KP10 集中参数法形式一致，只是这里 $\zeta_1$ 里含了内部导热的影响。

### 3.4 Heisler 图（诺谟图法）

工程上不解偏微分方程，而是查 **Heisler 图**（及配套的 位置修正图、Gröber 热量图）：

- **横轴 Fo、纵轴中心无量纲温度、参数 1/Bi** → 查中心温度随时间。
- 再用**位置图**（以 x/L 为参数）把中心温度换算到任意位置。
- 用 **Gröber 图**查已放出的热量占总热量的比例（算冷负荷、冻结放热量）。

三张图组合，覆盖平板、无限长圆柱、球三种基本几何。

### 3.5 与集中参数法的边界

- $\text{Bi} < 0.1$：内部温差可忽略 → 退化为 **KP10 集中参数法**（整块一个温度）。
- $\text{Bi} > 0.1$：必须用本知识点的 Heisler/级数解，**中心显著滞后**。

## 4. 冷库工程应用

1. **冷却/冻结时间估算**：给定货物尺寸、λ、ρ、c、风温与 h（Bi），用 Heisler 图查中心达标（如 −15 ℃）所需 Fo，反算时间 $t = \text{Fo}\cdot L_c^2/\alpha$。这是冷却间/冻结间设计的核心计算。
2. **码垛厚度与摆放**：$t \propto L_c^2$——货物特征尺寸翻倍，降温时间约变 **4 倍**。所以大块货物要么切分、要么延长冷却时间，这条平方律直接决定库容周转。
3. **中心温度是验收点**：食品安全看**中心温度**（最不利点），不是表面。非稳态导热告诉我们中心永远最后达标，验收必须测中心。
4. **α 的差异**：含水货物冻结后 λ 增大、α 增大，降温后段反而加快；而未冻结段 α 小、降温慢——解释了冻结曲线的"平台 + 折点"。

## 5. 典型例题（带数值）

**题目**：一块厚 $2L=0.1\,\text{m}$（半厚 $L_c=0.05\,\text{m}$）的瘦肉平板，初温 $T_i=20\,℃$，置于 $T_\infty=-25\,℃$ 冷风中，表面换热系数 $h=25\,\text{W/(m}^2\text{K)}$。肉的 $\lambda=0.5\,\text{W/(mK)}$、$\rho=1050\,\text{kg/m}^3$、$c=3500\,\text{J/(kgK)}$。求中心降到 $0\,℃$ 所需时间。

**解**：

1. 热扩散率：
$$\alpha = \frac{\lambda}{\rho c} = \frac{0.5}{1050\times 3500} \approx 1.36\times10^{-7}\,\text{m}^2/\text{s}$$

2. 毕渥数：
$$\text{Bi} = \frac{hL_c}{\lambda} = \frac{25\times0.05}{0.5} = 2.5\ (>0.1,\ \text{必须用非稳态})$$

3. 目标无量纲中心温度：
$$\theta^*_{center} = \frac{0-(-25)}{20-(-25)} = \frac{25}{45} \approx 0.556$$

4. 由 Bi=2.5（1/Bi=0.4）查平板 Heisler 图（或一项近似 $C_1\approx1.21,\ \zeta_1\approx1.14$）：
$$0.556 = 1.21\,e^{-1.14^2\,\text{Fo}} \Rightarrow e^{-1.30\,\text{Fo}} = 0.459 \Rightarrow \text{Fo} = \frac{-\ln 0.459}{1.30}\approx 0.60$$

5. 反算时间：
$$t = \frac{\text{Fo}\cdot L_c^2}{\alpha} = \frac{0.60\times0.05^2}{1.36\times10^{-7}} \approx \frac{0.0015}{1.36\times10^{-7}}\approx 1.10\times10^{4}\,\text{s}\approx 3.1\,\text{h}$$

**结论**：中心降到 0 ℃ 约需 **3 小时**。若把厚度翻倍（$L_c=0.1$），时间约变 4 倍到 12 h——直观体现 $t\propto L_c^2$。

## 6. 常见误区

- **误区一：用表面温度当验收温度。** 表面早就达标，中心还差得远。验收永远看中心（最不利点）。
- **误区二：以为降温时间与厚度成正比。** 实际是**平方关系**（$t\propto L_c^2$），厚度是降温时间的最敏感因素。
- **误区三：Bi 和 Fo 混为一谈。** Bi 是空间上的热阻比（管形状、管"中心滞后多严重"）；Fo 是无量纲时间（管进度）。一个定"形"，一个定"时"。
- **误区四：任何时候都能用一项近似。** 一项近似要求 Fo>0.2（度过初始急变段）；很早期须用完整级数或按半无限大物体（KP03）处理。
- **误区五：把 Bi 的 λ 用成流体的。** Bi 用**货物**导热系数 λ_goods；Nu 才用流体的 λ_fluid。

## 7. 与其他知识点的联系

- **← KP08 货物表面传热**：KP08 用 Bi 判断"要不要考虑内部滞后"，Bi>0.1 就分流到本页。表面能量平衡（第三类边界条件）是本页的边界条件。
- **↔ KP10 集中参数法**：本页是 Bi>0.1 的一般情形；当 Bi<0.1，中心不再滞后，退化为 KP10 的单一指数衰减。两者是"同一物理的两个极端"。
- **← KP03 半无限大物体导热**：非稳态降温的**极早期**（扰动还没到中心），货物就是半无限大物体，erf 解适用。KP03 是本页的"初始阶段极限"。
- **← KP06/KP04 对流**：表面换热系数 h（进而 Bi）来自强制/自然对流，是本页的输入。
- **→ 热负荷分析**：Gröber 图给出的放热量直接进入冷库冷负荷/冻结放热量计算。

## 8. 要点小结

| 要点 | 内容 |
|---|---|
| 核心方程 | $\partial T/\partial t=\alpha\nabla^2T$，α 是热扩散率 |
| 两个无量纲数 | Bi（空间·内外热阻比）、Fo（时间·无量纲时间） |
| 规则阶段 | Fo>0.2 时中心 $\theta^*=C_1e^{-\zeta_1^2\text{Fo}}$（指数衰减） |
| 工程工具 | Heisler 图 + 位置图 + Gröber 热量图（平板/柱/球） |
| 平方律 | 降温时间 $t\propto L_c^2$，厚度最敏感 |
| 验收点 | 中心温度（最不利点，最后达标） |
| 两端边界 | Bi<0.1→集中参数(KP10)；早期→半无限大(KP03) |
